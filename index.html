JAVA

JAVA

Java adalah bahasa pemrograman yang dapat dijalankan di komputer maupun smartphone. Bahasa ini dibuat pada tahun 1995 oleh James Gosling saat masih bergabung di Sun Microsystems. Java saat ini telah memiliki banyak rilisan, dan yang terbaru adalah Java SE 16. Aku pribadi mempelajari Java karena tuntutan dari Prodi Sistem Informasi di matkul Algoritma & Pemrograman.

HTML & CSS

HTML & CSS
HTML adalah kerangka yang membentuk dan mengatur bagaimana sebuah halaman web akan ditampilkan di browser. Untuk membentuk dan mengatur halaman tersebut kita menggunakan markah / tanda atau lebih umum disebut "tag". Ibaratnya HTML merupakan sebuah rumah namun belum terisi aksesoris yang cantik didalamnya. Oleh karena itu kita membutuhkan CSS untuk mengendalikan beberapa komponen dalam sebuah web sehingga tampilan akan lebih bagus dan terstruktur. Jika diibaratkan kembali, CSS seperti "dekorator" yang bertugas untuk mengisi rumah tadi dengan aksesoris yang bagus.

JAVASCRIPT

JAVASCRIPT
JavaScript adalah bahasa pemrograman yang digunakan dalam pengembangan website agar lebih dinamis dan interaktif. Kalau sebelumnya sudah membahas HTML dan CSS, nah si JavaScript ini dapat meningkatkan fungsionalitas pada halaman web. Selain membuat web jadi lebih dinamis dan interaktif, JS digunakan juga untuk proses logika data. “Intinya, fungsi JS tidak hanya soal urusan front end, tapi juga sekaligus dipakai untuk urusan back end,” -Kang Dea Afrizal

More About JavaScript

React JS [Front End]

React JS adalah sebuah pustaka / library javascript yang bersifat open source untuk membangun User Interface yang dibuat oleh Facebook. React JS mengurusi semua hal yang berkaitan dengan tampilan dan logika di sekitarnya. Popularitasnya dapat diukur oleh aplikasi yang menggunakannya seperti Facebook, WhatsApp, Netflix, Instagram, dan ratusan penyedia jasa pembuatan aplikasi berbasis web memanfaatkan kemampuan React JS. Express JS [Back End]

Express JS adalah framework dari NodeJS yang dirancang secara fleksibel dan sederhana untuk membantu tahap pengembangan aplikasi back end. Framework ini merupakan pengembangan dari NodeJS yang bertugas untuk menangani kebutuhan dalam server-side.
